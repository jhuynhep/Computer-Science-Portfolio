HTML/CSS

What is HTML?

HTML stands for HyperText Markup Language. This markup language is used to make webpages. In 1990, HTML was created by Tim Berners-Lee. HTML is used to add heading, paragraphs, images, and links to a webpage. In a HTML document, it is written by using HTML tags. HTML tags are used to descibe the content in a document. When creating an html document, the first line says . This is used in order to tell the browser that the document is in HTML. After writing up all of the codes, the text is saved as an html file and is then viewabale through a browser.





What is CSS?

CSS was created by the World Wide Web Consortium. CSS stands for cascading style sheets and CSS is used to style the webpage. CSS is used to style the page such as changing the font, creating a border, or adding special effectws to the page. Often times, there is a separate file for the css that is linked to the html file. This way, all of the codes used to style the page are all in one document.





What are tags?

Tags are keywords that help the browser know how to display the content. Tags are words that are surronded by angle brackets and there is an opening tag and a closing tag. However, in the closing tag, there is a slash before the tag name. In between an opening and a closing tag, the text between them is what appears on the website.

Examples of Tags

One example of a tag is <p>. This tag is used to start a paragraph. Another example of a tag is <h1>. This tag is used to create a heading for a page. An additional example of a tag is <ul> and this tag is used to start a list and indciate that it's an orderlist. One more example of a tag is <br>. This tag is used to indicate a line break.





What are attributes?

Attributes are placed inside the starting tag, after the selector but before the value. The purpose of the attribute is to provide addition information about the element. For instance, you could use the attribute of class="specificword" in order to later style that element.
