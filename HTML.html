HTML/CSS

What is HTML?

HTML stands for HyperText Markup Language. This markup language is used to make webpages. In 1990, HTML was created by Tim Berners-Lee, but the reason html was created is still unclear. HTML is used to add heading, paragraphs, images, and links to a webpage. In a HTML document, it is written by using HTML tags. HTML tags are used to descibe the content in a document. When creating an html document, the first line says <!DOCTYPE html>. This is used in order to tell the browser that the document is in HTML. After writing up all of the codes, the text is saved as an html file and is then viewabale through a browser.





What is CSS?

CSS was created by the World Wide Web Consortium because when the font tag and color attributes were first created, developers had to add those properties to every single tag. Therefore, CSS was introduced to make it easier and faster to style the page. CSS stands for cascading style sheets and CSS is used to style the webpage. CSS is used to style the page such as changing the font, creating a border, or adding special effects to the page. Often times, there is a separate file for the css that is linked to the html file. This way, all of the code used to style the page are all in one document.





What are tags?

Tags are keywords that help the browser know how to display the content. Tags have to go through the web browser in order for the webpage to be displayed on the screen. Another way to say this would be that tags are a piece of code that identifies the starts or end of an element. Tags are words that are surronded by angle brackets and there is an opening tag and a closing tag. However, in the closing tag, there is a slash before the tag name. In between an opening and a closing tag, the text between them is what appears on the website. The tag text itself does not apeear on the webpage.

Examples of Tags
Tag Function Characteristic
<p> The p tag is used to start a paragraph. Defaultly, it is a block element, meaning that the space that the p tag takes up will naturally expand as you add more words to the paragrph.
<h1> The h1 tag is used to create a heading for a page It will usually be the largest text on the webpage.
<ul> The ul tag is used to start a list and indicate that it's an unordered list Because it's an unordered list, it's in bullet point form.
<br> The br tag is used to indicate a line break The br tag is an empty tag, meaning that it had no end tag.




What are attributes?

Attributes are placed inside the starting tag, after the tag name but before the value. The purpose of the attribute is to provide additional information, or modify the element. For instance, you could use the attribute of class in order to later style that element using css.
