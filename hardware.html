Hardware

Parts of a Computer

RAM stands for random access memory. RAM is a type of computer memory that can be accessed randomly. It's a hardware device that allows information to be stored or retrieved on the computer. RAM requires power to keep the data accessible.

CPU stands for central processing unit. It's like the brains for a computer and the CPU is where most calculations take place. Modern CPUs are small, square and also contain mutiple metallic connectors or pins on the underside. Then the CPU is inserted into a CPU socket that is located on the motherboard.
